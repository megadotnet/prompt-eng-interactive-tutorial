# 附录 10.3：搜索与检索

您知道您可以使用 Claude **为您搜索维基百科**吗？Claude 可以查找和检索文章，此时您还可以使用 Claude 来总结和综合它们，根据找到的内容编写新颖的内容等等。不仅仅是维基百科！您还可以搜索自己的文档，无论是存储为纯文本还是嵌入到矢量数据存储中。

请参阅我们的 [RAG 食谱示例](https://github.com/anthropics/anthropic-cookbook/blob/main/third_party/Wikipedia/wikipedia-search-cookbook.ipynb)，了解如何补充 Claude 的知识，并利用从矢量数据库、维基百科、互联网等检索到的数据来提高 Claude 响应的准确性和相关性。在那里，您还可以了解如何使用某些[嵌入](https://docs.anthropic.com/claude/docs/embeddings) 和矢量数据库工具。

如果您有兴趣了解使用 Claude 的高级 RAG 架构，请查看我们的 [关于 RAG 架构的 Claude 3 技术演示幻灯片](https://docs.google.com/presentation/d/1zxkSI7lLUBrZycA-_znwqu8DDyVhHLkQGScvzaZrUns/edit#slide=id.g2c736259dac_63_782)。
