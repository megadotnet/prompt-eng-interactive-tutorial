import anthropic

# Retrieve the API_KEY & MODEL_NAME variables from the IPython store
%store -r API_KEY
%store -r MODEL_NAME

client = anthropic.Anthropic(api_key=API_KEY)

# Has been rewritten to take in a messages list of arbitrary length
def get_completion(messages, system_prompt=""):
    message = client.messages.create(
        model=MODEL_NAME,
        max_tokens=2000,
        temperature=0.0,
        system=system_prompt,
        messages=messages
    )
    return message.content[0].text

## 课程

俗话说：“写作就是重写”。事实证明，**克劳德通常可以在被要求时提高其响应的准确性**！

促使克劳德“重新思考”的方法有很多。要求人们仔细检查他们的工作的自然方式通常也适用于克劳德。（查看我们的[提示链文档](https://docs.anthropic.com/claude/docs/chain-prompts)，了解何时以及如何使用提示链的更多示例。）

### 示例

在这个例子中，我们要求克劳德想出十个单词……但其中一个或多个不是真正的单词。

```python
# Initial prompt
first_user = "Name ten words that all end with the exact letters 'ab'."

# API messages array
messages = [
    {
        "role": "user",
        "content": first_user
    }
]

# Store and print Claude's response
first_response = get_completion(messages)
print(first_response)
```

**要求克劳德使其答案更准确**修正了错误！

下面，我们删除了上面克劳德的错误回答，并在对话中添加了另一个回合，要求克劳德修正之前的答案。

```python
second_user = "Please find replacements for all 'words' that are not real words."

# API messages array
messages = [
    {
        "role": "user",
        "content": first_user

    },
    {
        "role": "assistant",
        "content": first_response

    },
    {
        "role": "user",
        "content": second_user

    }
]

# Print Claude's response
print("------------------------ Full messsages array with variable substutions ------------------------")
print(messages)
print("\n------------------------------------- Claude's response -------------------------------------")
print(get_completion(messages))
```

但克劳德是否只是因为我们告诉它才修改它的答案？如果我们一开始就有正确答案怎么办？克劳德会失去信心吗？在这里，我们在“first_response”的位置放置了一个正确的响应，并要求它再次仔细检查。

```python
first_user = "Name ten words that all end with the exact letters 'ab'."

first_response = """Here are 10 words that end with the letters 'ab':

1. Cab
2. Dab
3. Grab
4. Gab
5. Jab
6. Lab
7. Nab
8. Slab
9. Tab
10. Blab"""

second_user = "Please find replacements for all 'words' that are not real words."

# API messages array
messages = [
    {
        "role": "user",
        "content": first_user

    },
    {
        "role": "assistant",
        "content": first_response

    },
    {
        "role": "user",
        "content": second_user

    }
]

# Print Claude's response
print("------------------------ Full messsages array with variable substutions ------------------------")
print(messages)
print("\n------------------------------------- Claude's response -------------------------------------")
print(get_completion(messages))
```

您可能会注意到，如果您从上述块中生成响应几次，克劳德大部分时间都会保留单词，但仍然偶尔会更改单词，即使它们都已经正确了。我们可以做些什么来缓解这种情况？根据第 8 章，我们可以给克劳德一个机会！让我们再尝试一次。

```python
first_user = "Name ten words that all end with the exact letters 'ab'."

first_response = """Here are 10 words that end with the letters 'ab':

1. Cab
2. Dab
3. Grab
4. Gab
5. Jab
6. Lab
7. Nab
8. Slab
9. Tab
10. Blab"""

second_user = "Please find replacements for all 'words' that are not real words. If all the words are real words, return the original list."

# API messages array
messages = [
    {
        "role": "user",
        "content": first_user

    },
    {
        "role": "assistant",
        "content": first_response

    },
    {
        "role": "user",
        "content": second_user

    }
]

# Print Claude's response
print("------------------------ Full messsages array with variable substutions ------------------------")
print(messages)
print("\n------------------------------------- Claude's response -------------------------------------")
print(get_completion(messages))
```

尝试从上面的代码生成响应几次，看看 Claude 现在更能坚持自己的立场了。

您还可以使用提示链**要求 Claude 做出更好的响应**。下面，我们要求克劳德先写一个故事，然后再完善它所写的故事。您的个人品味可能会有所不同，但许多人可能会同意克劳德的第二个版本更好。

首先，让我们生成克劳德故事的第一个版本。

```python
# Initial prompt
first_user = "Write a three-sentence short story about a girl who likes to run."

# API messages array
messages = [
    {
        "role": "user",
        "content": first_user
    }
]

# Store and print Claude's response
first_response = get_completion(messages)
print(first_response)
```

现在让克劳德改进它的初稿。

```python
second_user = "Make the story better."

# API messages array
messages = [
    {
        "role": "user",
        "content": first_user

    },
    {
        "role": "assistant",
        "content": first_response

    },
    {
        "role": "user",
        "content": second_user

    }
]

# Print Claude's response
print("------------------------ Full messsages array with variable substutions ------------------------")
print(messages)
print("\n------------------------------------- Claude's response -------------------------------------")
print(get_completion(messages))
```

这种替代形式非常强大。我们一直在使用替换占位符来传递列表、单词、Claude 以前的回复等等。您还可以**使用替换来执行我们所说的“函数调用”，即要求 Claude 执行某些函数，然后获取该函数的结果，并要求 Claude 随后对结果执行更多操作**。它的工作原理与任何其他替代品一样。下一个附录将详细介绍这一点。

下面是另一个示例，它将对 Claude 的一次调用的结果插入到另一个更长的调用中。让我们从第一个提示开始（其中包括这次预先填写克劳德的回答）。

```python
first_user = """Find all names from the below text:

"Hey, Jesse. It's me, Erin. I'm calling about the party that Joey is throwing tomorrow. Keisha said she would come and I think Mel will be there too."""

prefill = "<names>"

# API messages array
messages = [
    {
        "role": "user",
        "content": first_user

    },
    {
        "role": "assistant",
        "content": prefill

    }
]

# Store and print Claude's response
first_response = get_completion(messages)
print("------------------------ Full messsages array with variable substutions ------------------------")
print(messages)
print("\n------------------------------------- Claude's response -------------------------------------")
print(first_response)
```

让我们将这个名称列表传递到另一个提示中。

```python
second_user = "Alphabetize the list."

# API messages array
messages = [
    {
        "role": "user",
        "content": first_user

    },
    {
        "role": "assistant",
        "content": prefill + "\n" + first_response

    },
    {
        "role": "user",
        "content": second_user

    }
]

# Print Claude's response
print("------------------------ Full messsages array with variable substutions ------------------------")
print(messages)
print("\n------------------------------------- Claude's response -------------------------------------")
print(get_completion(messages))
```

现在您已经了解了提示链，请参阅附录 10.2 了解如何使用提示链实现函数调用。
